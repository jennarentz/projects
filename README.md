| [Neighborly](./neighborly) | A social app where users can post and view events, clubs, and more within their neighborhood.  |
| [Personal Finance Tracker](./personal-finance-tracker) | A Streamlit app for uploading and analyzing personal finance transactions. |

## Group Project

### [ClassGPT - AI Classwork Chatbot](https://github.com/CS222-UIUC/fa25-team110-classgpt)

ClassGPT is a group project created for CS 222 at UIUC. It is an AI-powered academic chatbot designed to help students ask questions about class materials while allowing professors to control the content the chatbot uses. The project uses a Streamlit frontend, Django backend, and Ollama-powered AI layer. Professors can upload course materials, and students can ask questions that are answered based on those materials.

## How to Use

1. Clone the repository.

```bash
git clone https://github.com/jennarentz/projects.git
cd projects
```

2. Open the folder for the project you want to run.

```bash
cd movie-explorer
```

3. Follow that project’s individual README instructions.
